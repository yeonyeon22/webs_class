CSS Padding

Padding

padding 속성은 요소의 안쪽에 배치되는 여백이며 요소의 경계선과 내용물 사이에 삽입되는 빈 공간이다.
요소의 크기에 포함된다.

정의

  • padding-top: 요소의 위쪽 패딩값
  • padding-right: 요소의 오른쪽 패딩값
  • padding-bottom: 요소의 아래쪽 패딩값
  • padding-left: 요소의 왼쪽 패딩값
  • padding: 10px; /* 위,오른쪽,아래,왼쪽 : 10px */
  • padding: 10px 11px; /* 위,아래:10px 왼쪽,오른쪽:11px */
  • padding: 10px 11px 12px; /* 위:10px 왼쪽,오른쪽:11px 아래:12px */
  • padding: 10px 11px 12px 13px; /* 위:10px 오른쪽:11px 아래:12px 왼쪽:13px */
