1. 1부터 100까지 출력하기

1.1 while문을 이용해서 1부터 100까지 출력하는 함수를 작성해주세요.

{
    let num = 1;

    while(num <= 100){
        console.log(num);
        num++;
    }
}

    for(let i=1; i<=100; i++){
        console.log(i+);
    }
}

for문을 이용해서 1부터 100까지 출력하는 함수를 작성해주세요.

조건1. 짝수만 출력하세요!

조건2. 홀수만 출력하세요!

{
    //짝수
    for(let i=1; i<=100; i++){
        console.log(i);
        if( i % 2 == 0){
            console.log(i);
        }
    }

    //홀수
    for(let i=1; i<=100; i++){
        console.log(i);
        if( i % 2 != 0){
            console.log(i);
        }
    }
}
