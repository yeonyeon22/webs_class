HTML 레퍼런스 속성과 설명을 나타내는 표입니다.
설명 속성
<a> <a> 태그는 하나의 페이지에서 다른 페이지를 연결할 때 사용하는 하이퍼링크(hyperlink)를 정의할 때 사용합니다.
<abbr> <abbr> 태그는 “W3C”, “HTML”, “ASAP” 등과 같이 단어의 축약형(abbreviation)이나 머리글자로만 된 단어(acronym)를 정의할 때 사용합니다.
<acronym> <acronym> 태그는 더 이상 사용하지 않고, 사용해서도 안되는 오래된 HTML 요소입니다.
<address> <address> 태그는 문서나 글의 저자 또는 회사와 연락할 수 있는 정보를 명시할 때 사용합니다.
<applet> <applet> 태그는 더 이상 사용하지 않고, 사용해서도 안되는 오래된 HTML 요소입니다.
<area> <area> 태그는 이미지 맵(image-map)에서 하이퍼링크가 연결될 영역을 정의할 때 사용합니다.
<article> <article> 태그는 해당 문서나 페이지 또는 사이트와는 완전히 독립적으로 구성할 수 있는 요소를 정의할 때 사용합니다.
<aside> <aside> 태그는 문서의 주요 내용과 간접적으로만 연관된 부분을 나타냅니다. 주로 사이드바 혹은 콜아웃 박스로 표현합니다.
<audio> <audio> 태그는 음악이나 오디오 스트림과 같은 사운드를 정의할 때 사용합니다.
<b> <b> 태그는 다른 부가적인 목적 없이 단순히 굵게 표현되는 텍스트를 정의할 때 사용합니다.
<base> <base> 태그는 해당 문서의 모든 상대 주소(relative URL)에 대한 기본 URL(base URL)과 target 속성값을 정의할 때 사용합니다.
<basefont> <basefont> 태그는 더 이상 사용하지 않고, 사용해서도 안되는 오래된 HTML 요소입니다.
<bdi> <bdi> 태그는 Bi-Directional Isolation을 의미하며, 주위의 텍스트들과는 다른 방향으로 써지는 텍스트 영역을 정의할 때 사용합니다.
<bdo> <bdo> 태그는 Bi-Directional Override를 의미하며, 자식 요소의 텍스트 방향성(text directionality)을 정의할 때 사용합니다.
<big> <big> 더 이상 사용하지 않고, 사용해서도 안되는 오래된 HTML 요소입니다.
<blockquote> <blockquote> 태그는 안쪽의 텍스트가 긴 인용문임을 나타냅니다. 주로 들여쓰기를 한 것으로 그려집니다.
<blink> <blink> 태그는 더 이상 사용하지 않고, 사용해서도 안되는 오래된 HTML 요소입니다.
<body> 태그는 문서 본문에 해당하는 콘텐츠를 포함하는 영역입니다.
<br> <br> 태그는 텍스트를 줄바꿈 할 때 설정합니다.
<button> <button> 태그는 클릭할 수 있는 버튼을 정의할 때 사용합니다.
<canvas> <canvas> 태그는 주로 자바스크립트와 같은 스크립트를 이용하여 그래픽 콘텐츠를 그릴 때 사용합니다.
<caption> <caption> 태그는 태그는 테이블의 캡션(caption, 테이블이나 사진, 삽화 등에 붙는 설명)을 정의할 때 사용합니다.
<center> <center> 태그는 더 이상 사용하지 않고, 사용해서도 안되는 오래된 HTML 요소입니다.
<cite> <cite> 태그는 책이나 음악, 영화, 예술 작품 등과 같은 창작물의 제목을 정의할 때 사용합니다.
<code> <code> 태그는 코드를 나타내는 텍스트를 설정합니다.
<col> <col> 태그는 <colgroup> 요소에 속하는 각 열의 속성을 설정합니다.
<colgroup> <colgroup> 태그는 테이블에서 서식 지정을 위해 하나 이상의 열을 그룹으로 묶을 때 사용합니다.
<data> <data> 태그는 특정 콘텐츠에 기계가 읽을 수 있는(machine-readable) 형태의 값(value)을 덧붙여 나타낼 때 사용합니다.
<datalist> <datalist> 태그는 <input> 요소에서 사용하기 위한 옵션들의 리스트를 미리 정의할 때 사용합니다.
<dd> <dd> 태그는 정의 목록 요소(dl)에서 앞선 용어(dt)에 대한 설명, 정의, 또는 값을 제공합니다.
<del> <del> 태그는 텍스트 한가운데 라인(line)을 추가하여 문서에서 삭제된 텍스트를 표현할 때 사용합니다.
<details> <details> 태그는 사용자가 직접 조작하여 보거나 숨길 수 있는 부가적인 세부사항(additional details)을 명시할 때 사용합니다.
<dfn> <dfn> 태그는 HTML에서 용어(term)의 정의를 나타낼 때 사용합니다.
<dialog> <dialog> 태그는 대화 상자(dialog box)나 대화 윈도우(dialog window)를 정의할 때 사용합니다.
<dir> <dir> 더 이상 사용하지 않고, 사용해서도 안되는 오래된 HTML 요소입니다.
<div> <div> 태그는 문서의 영역을 나누거나 그룹화 할 때 설정합니다.
<dl> <dl> 태그는 설명 목록을 나타냅니다.
<dt> <dt> 태그는 설명 혹은 정의 리스트에서 용어를 나타냅니다. dl 요소 안에 위치해야 합니다
<em> <em> 태그는 텍스트를 강조할 때 설정합니다.
<embed> <embed> 태그는 외부 어플리케이션이나 대화형 컨텐츠와의 통합점을 나타냅니다.
<fieldset> <fieldset> 태그는 웹 양식의 여러 컨트롤과 레이블(label)을 묶을 때 사용합니다.
<figcaption> <figcaption> 태그는 부모 figure 요소가 포함하는 다른 콘텐츠에 대한 설명 혹은 범례를 나타냅니다.
<figure> < ;figure> 태그는 독립적인 콘텐츠를 표현합니다. figcaption 요소를 사용해 설명을 붙일 수 있습니다.
<font> <font> 더 이상 사용하지 않고, 사용해서도 안되는 오래된 HTML 요소입니다.
<footer> <footer> 가장 가까운 구획 콘텐츠나 구획 루트의 푸터를 나타냅니다. 푸터는 일반적으로 구획의 작성자, 저작권 정보, 관련 문서 등의 내용을 담습니다.
<form> <form> 태그는 사용자로부터 입력을 받을 수 있는 HTLM 입력 폼(form)을 정의할 때 사용합니다.
<frame> <frame> 태그는 더 이상 사용하지 않고, 사용해서도 안되는 오래된 HTML 요소입니다.
<frameset> <frameset> 태그는 더 이상 사용하지 않고, 사용해서도 안되는 오래된 HTML 요소입니다.
<h1> ~ <h6> <h1> ~ <h6> 태그는 섹션의 제목을 설정합니다.
<head> <head> 태그는 제목과 스크립트, 스타일 시트, 메타 정보 등을 제공하는 영역입니다.
<header> <header> 태그는 소개 및 탐색에 도움을 주는 콘텐츠를 나타냅니다. 제목, 로고, 검색 폼, 작성자 이름 등의 요소도 포함할 수 있습니다.
<hgroup> <hgroup> 태그는 더 이상 사용하지 않고, 사용해서도 안되는 오래된 HTML 요소입니다.

<html> <html> 태그는 웹 문서의 최상위 요소 문서의 루트를 나타냅니다.
<i> <i> 태그는 기본 텍스트와는 다른 분위기나 음성을 위한 텍스트 영역을 정의할 때 사용합니다.
<iframe> <iframe> 태그는 인라인 프레임(inline frame)을 정의할 때 사용합니다.
<img> <img> 태그는 웹 문서에 이미지를 설정합니다.
<input> 태그는 사용자로부터 입력을 받을 수 있는 입력 필드(input filed)를 정의할 때 사용합니다.
<ins> <ins> 태그는 텍스트 아래쪽에 라인(line)을 추가하여 문서에 추가된 텍스트를 표현할 때 사용합니다.
<kbd> <kbd> 태그는 키보드(keyboard) 입력을 나타낼 때 사용합니다.
<keygen> <keygen> 태그는 더 이상 사용하지 않고, 사용해서도 안되는 오래된 HTML 요소입니다.
<label> <label> 태그는 사용자 인터페이스(UI) 요소의 라벨(label)을 정의할 때 사용합니다.
<legend> <legend> 태그는 <fieldset> 요소의 캡션(caption)을 정의할 때 사용합니다.
<li> <li> 태그는 목록에 항목을 설정합니다.
<link> <link> 태그는 외부 파일을 연결할 때 설정합니다.
<main> <main> 태그는 웹 문서의 주요 콘텐츠 영역을 설정합니다.
<map> <map> 태그는 클라이언트 사이드 이미지맵(client-side image-map)을 정의할 때 사용합니다.
<mark> <mark> 태그는 형광펜으로 칠한 것처럼 하이라이트된(highlighted) 텍스트를 정의할 때 사용합니다.
<marquee> <marquee> 태그는 더 이상 사용하지 않고, 사용해서도 안되는 오래된 HTML 요소입니다.

<meta> <meta> 태그는 해당 문서에 대한 정보인 메타데이터(metadata)를 정의할 때 사용합니다.
<meter> <meter> 태그는 분수 값이나 게이지(gauge) 내에서 특정 스칼라 값(scalar measurement)이 어느 정도인지를 표현할 때 사용합니다.
<nav> <nav> 태그는 문서의 부분 중 현재 페이지 내, 또는 다른 페이지로의 링크를 보여주는 구획을 나타냅니다.
<noframes> <noframes> 태그는 더 이상 사용하지 않고, 사용해서도 안되는 오래된 HTML 요소입니다.

<object> <object> 태그는 외부 리소스의 종류에 따라 문서 내에서 처리하는 방식을 변경할 수 있는 임베디드 객체(embedded object)를 정의할 때 사용합니다.
<ol> <ol> 태그는 순서가 있는 HTML 리스트(list)를 정의할 때 사용합니다.

<title> <title> 태그는 브라우저의 제목 또는 탭에 표시되는 문서의 제목을 설정합니다.
<tt> <tt> 태그는 더 이상 사용하지 않고, 사용해서도 안되는 오래된 HTML 요소입니다.
<u> <u> 태그는 철자가 틀린 단어나 중국어의 고유 명사처럼 문체상 일반적인 텍스트와는 달라야만 하는 텍스트를 표현할 때 사용합니다.
<ul> <ul> 태그는 순서가 없는 HTML 리스트(list)를 정의할 때 사용합니다.
<var> <var> 태그는 변수(variable)를 정의할 때 사용합니다.
<video> <video> 태그는 무비 클립(movie clip)이나 비디오 스트림(video stream)과 같은 비디오를 정의할 때 사용합니다.
<wbr> <wbr> 태그(Word Break Opportunity)는 단어 중간에서 행바꿈될 수 있는(line-break) 위치를 정의할 때 사용합니다.
